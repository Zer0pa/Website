# Zer0pa — Onboarding Landscape

> Generated: 2026-03-27 · Phase 1 Onboarding

## Overview

**Zer0pa** builds a family of **deterministic, CPU-native codecs** under the umbrella brand **Zero-Point Encoding (ZPE)**. There are **10 public repos** on GitHub under [`github.com/Zer0pa`](https://github.com/Zer0pa), each targeting a distinct signal modality. All share:

- **Zer0pa SAL v6.0** license (free ≤ USD 100M revenue)
- No GPU, no neural network, no training loop
- Proof-custody architecture (retained evidence bundles)
- Auditor playbooks & public audit limits
- Contact: `architects@zer0pa.ai` · Site: `www.zer0pa.ai`

## Content Strategy Principles

1. **Copy comes from the repos** — README content is the source of truth; we don't write independent marketing copy
2. **Metrics are proof-bounded** — we only promote what the proof bundles support
3. **Honest non-claims** — repos explicitly state what is NOT proven; website should mirror this integrity
4. **GitHub as living documentation** — website links directly to repo docs, not duplicated prose
5. **Deterministic = the brand** — no GPU, no neural network, no training loop — this is the core differentiator
