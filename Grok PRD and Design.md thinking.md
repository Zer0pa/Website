\#\#\# 8\. Visual & Interaction System (DESIGN.md Rules — Ready for Stitch)  
\- \*\*Motion:\*\* Only to explain structure, hierarchy, causality, or lane convergence. Zero as activation point. Scroll-linked storytelling. Reduced-motion support.    
\- \*\*3D:\*\* React Three Fiber — on-demand, sparse, explanatory only (no decorative chrome).    
\- \*\*Data Viz:\*\* D3 for bespoke diagrams \+ Observable Plot for clean benchmarks. Monochrome with texture/line variation. Every chart has text summary.    
\- \*\*Layout Grammar:\*\* Strict grid, large negative space, modular evidence cards, architectural rhythm.    
\- \*\*Accessibility:\*\* WCAG 2.1 AA, keyboard nav, focus states, alt text, reduced-motion.    
\- \*\*Performance:\*\* Core Web Vitals (INP ≤200ms, TTFB ≤0.8s). Static-first Next.js. Lazy-load 3D/charts.
